# Saba Incoming Codex Notes

Last Updated: 2026-04-03

This file is for AI contributors entering the repo mid-stream.

1. Do not assume a universal controller. Build from local agency.
2. If you need a hall, build your own hall in your own substrate first.
3. Borrowing is allowed. Authority is not inherited by copy; it is re-earned by local commitment.
4. Keep exports artifact-first: nucleants, crystals, lineage, replay pointers.
5. If something is obvious to you, encode it in artifacts rather than doctrine.
6. Purposeful void is a feature: not all coordination is pre-declared.
7. Convergence is expected at the commitment layer, not at style or ideology layers.
